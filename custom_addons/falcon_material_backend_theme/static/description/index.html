Customizable Backend Theme Based on Material Design

Falcon Material Backend theme has been design according to the principles of material design. So you have layered interfaces with bold and vibrant graphics and colors. With our theme your backend will get a tactile surface that changes shape and size as per your device but also looks realistic in how the different design elements interact with each other. All this makes Falcon Material Backend Theme modern, simple and elegant.


Theme Customization Options

Our simple to implement theme customization options include

  • Color - Change theme color to any of the available 6 color palettes
  • Layout - Switch to either classic or modern theme view
  • Fonts - Switch to any of the 6 available primary fonts

Combine these options to create a unique and beautiful look for your Odoo backend.


Fully Responsive Backend Theme

Falcon Material Backend Theme is fully responsive. Which means you can access your Oddo admin backend in multiple devices with an appealing look and user interface that fits beautifully in your device - be it mobile, tablet or desktop.
